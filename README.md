# 💊 AI-Enhanced Pharmacy Management System

A Python Tkinter-based Pharmacy Management System with separate Admin and Employee panels, inventory and billing management, medicine CRUD operations (Add, Update, Delete, Save, Search), API-based medicine search, and AI-powered low-stock and expiry alerts for efficient pharmacy operations.

## ✨ Features

- 👨‍💼 Admin & Employee role system
- 💊 Medicine management (Add, Update, Delete, Save, Search)
- 📦 Inventory tracking system
- 🧾 Billing and sales system
- ⚠️ AI-powered low stock alerts
- ⏰ Expiry date monitoring
- 🔎 API-based medicine search integration

## 🛠️ Tech Stack

- Python
- Tkinter
- SQLite (if used)
- APIs
- OOP Concepts
